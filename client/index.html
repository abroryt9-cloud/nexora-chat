Условия использования Nexora

1. Общие положения

Nexora - это современный мессенджер, предоставляющий возможность обмена сообщениями, медиафайлами и голосовыми сообщениями. Используя сервис, вы соглашаетесь с данными условиями.

2. Конфиденциальность

Мы уважаем вашу конфиденциальность. Все сообщения шифруются и хранятся только на вашем устройстве. Мы не передаем данные третьим лицам.

3. Правила поведения

Запрещается: спам, оскорбления, распространение запрещенного контента, нарушение авторских прав, мошенничество.

4. Ответственность

Пользователь несет ответственность за все действия, совершенные в аккаунте. Администрация не несет ответственности за содержание сообщений.

5. Изменения условий

Мы оставляем право изменять условия использования. О важных изменениях мы уведомим вас по email.

6. Контакты

По вопросам: support@nexora.com

Политика конфиденциальности

Какие данные мы собираем?

• Имя пользователя и email
• Аватар (по вашему выбору)
• История сообщений (только на вашем устройстве)
• Технические данные (IP, браузер)

Как мы используем данные?

• Для обеспечения работы мессенджера
• Для улучшения сервиса
• Для связи с вами по важным вопросам

Безопасность

Мы используем современные методы шифрования для защиты ваших данных. Ваши сообщения доступны только вам и вашим собеседникам.
