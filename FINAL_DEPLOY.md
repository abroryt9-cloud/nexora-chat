### 1. Откройте мессенджер

```
https://nexora-chat.onrender.com
```

**Должен открыться:**
- ✨ Экран входа (не JSON!)
- 🎨 Чёрный фон с бирюзовыми акцентами

### 2. Войдите в чат

1. Введите имя (например, `Alex`)
2. Нажмите "Войти в чат"
3. Должен открыться экран чата

### 3. Отправьте сообщение

1. Введите текст сообщения
2. Нажмите "Отправить" или Enter
3. Сообщение должно появиться в чате (зелёное, справа)

### 4. Проверьте WebSocket

1. Откройте консоль браузера (F12)
2. Посмотрите логи:
   ```
   [WebSocket] Connected: <socket_id>
   [WebSocket] Auth OK: {...}
   ```

### 5. Тест с другим пользователем

1. Откройте сайт в другом браузере/вкладке
2. Введите другое имя (например, `Maria`)
3. Отправьте сообщение
4. В первом браузере должно появиться сообщение (серое, слева)

## 🎯 Как это работает

### Архитектура

```
┌──────────────────────────────────────────┐
│  Browser (Client)                        │
│  ┌────────────────────────────────────┐  │
│  │  public/index.html                 │  │
│  │  - Login Screen                    │  │
│  │  - Chat Screen                     │  │
│  │  - WebSocket Client                │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
              ↕ WebSocket (Socket.io)
┌──────────────────────────────────────────┐
│  Render Server                           │
│  ┌────────────────────────────────────┐  │
│  │  server.js                         │  │
│  │  - Express (static files)          │  │
│  │  - Socket.io (WebSocket)           │  │
│  │  - Message broadcast               │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

### Поток сообщений

```
User A: "Привет!"
   ↓ (emit)
socket.emit('message', { text: "Привет!" })
   ↓ (server broadcast)
io.emit('message', data)
   ↓ (receive)
User B получает сообщение
User A получает своё сообщение
```

## 🐛 Устранение проблем

### Проблема: Открывается JSON вместо интерфейса

**Причина:** Файл `public/index.html` не найден

**Решение:**
```bash
# Проверьте, что файл существует
ls public/index.html

# Если нет - создайте
# (файл уже создан в этом коммите)

# Закоммитьте и отправьте
git add public/index.html
git commit -m "Add public/index.html"
git push origin main
```

### Проблема: WebSocket не подключается

**Причина:** Неправильный URL или CORS

**Решение:**
1. Откройте консоль браузера (F12)
2. Посмотрите ошибки
3. Проверьте, что используете HTTPS

### Проблема: Сообщения не отправляются

**Причина:** WebSocket не подключён

**Решение:**
1. Проверьте индикатор подключения (должен быть зелёным)
2. Если красный - перезагрузите страницу

### Проблема: Кэш браузера

**Решение:**
```
Ctrl + Shift + R (Windows)
Cmd + Shift + R (Mac)
```

Или откройте в режиме инкогнито.
